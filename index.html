DDB Redirecter

Welcome to the DDB Redirecter! I hope you find what you're looking for.

Message from the other side...

What is this?

A self-contained redirecter to D&D Beyond. Enter a source and a page number and you'll redirected to the (closest) corresponding section on DDB.

What does it mean it's self-contained? All the code is within the page itself, with the exception of the stylesheet.

Why is it?

I'm a D&D newbie and started "all digital" with D&D Beyond. I found myself wanting to look up these page references that kept coming up here and there, but there wasn't an easy way of doing this. This isn't a huge problem, but I was interested to see if I could make a page reference mapped to DDB.

I started out by making a DDB to physical page reference, meaning I went from the table of contents on DDB and checked which physical pages they represented.

It wasn't an optimal solution, so I did this thing instead! This redirecter can be used in all kind of places. You can even add it as a Custom Search Engine in Chrome!

How do I use it?

Add the source book acronym and page number to this page's URL. For instance, if you want to go to Player's Handbook page 256, add ?PHB256 at the end of the URL. Or use the form above!

Notes on PHB
In the physical book, the section on Weapon describes the weapon properties and is followed with a table showing the common weapons. On DDB these are the other way around, so the content flow is slightly different.
The pages following are on Adventuring Gear, and all link to the section Adventuring Gear on DDB, since the individual items are non-linkable.
Notes on MM
Some statblocks are linkable and some are not. The flow will be off sometimes, due to the physical book sometimes having statblocks on an entire page.
Notes on DMG
Chapter 7 is split up in several pages due to the vast number of magic items etc. This means if you search for the page that is in the middle between these sections, you won't be able to scroll to the right place (they're on different base pages). If you find that this is the case, just move one more page ahead. The pages in Appendix C all link to the beginning of the appendix, since there are no linkable sections. Some page in Appendix B suffer the same fate.

            
How precise is it?

            
The redirecter is dependent on being able to link to the different sections on DDB. Not all sections are linkable, though.

            
Many pages in the physical books begin with a new section, and thus I can link to the precise place of the page break, provided DDB has a linkable section.

            
Physical book pages that does not begin with a new section, are linked to the closest or most logical corresponding section on DDB.


            
Exact page break

            
Example of exact page breaks are the beginning of new chapters, races, classes etc. If a page start with a new section, I can almost always link to that exact section.


            
Non-exact page breaks

            
Many pages in the physical books start with a bit of text from the preceding section. In this case, I sometimes link to the section that started on the page 
                before or the first section on the new page. In either case, I indicate that the physical page break was above or below the 
            section I sent you to. This indication shows up in the URL as ?above or ?below followed with the section ID.

            
If the content on the page starts with maybe half a sentence, or some text that is very unlikely to be referenced, I don't indicate at all.


            
Pages with images

            
Some pages in the physical books consists of only an image. Most of this art is presented somewhere on DDB, but I choose not to redirect to these images.
                One exception are the covers of the books.
